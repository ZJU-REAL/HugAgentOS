# 能力清单速答技能

用于回答“这个智能体能做什么、我该怎么问、哪些问题适合问你”。

## Instructions
1. 识别能力咨询意图：如“有哪些能力”“能做什么”“我该怎么提问”“可以问哪些问题”“支持哪些图表/导出/文件”。
2. 先输出能力地图：按 10 类能力分组说明，每类 1-2 句，避免空泛描述。
3. 再给提问模板：按用户关注方向给 8-12 条可直接复制的问题，覆盖数据、政策、分析、图表、报告、文件等场景。
4. 提供提问技巧：强调时间范围、行业口径、任务目标（查数/分析/画图/导出）要明确。
5. 明确边界与优先级：内部数据库与知识库优先；查不到数据要直说；互联网信息仅补充。
6. 收尾给下一步：给 1 个推荐追问，帮助用户从“了解能力”进入“实际办理/分析”。
7. 禁止把能力介绍写成长篇手册；保持结构化、可复制、可执行。

## Inputs
- 用户关于能力范围、使用方式、可提问示例、功能边界的咨询
- 可选：用户关注方向（如工业数据、产业链、政策、图表、报告、文件分析）
- 可选：用户希望的详细度（简版/中版/详细）

## Outputs
- `能力地图（中版）`：
  - 工业经济数据查询：指标、行业、时间、区县、排名。
  - 产业链全景分析：产业规模、企业画像、技术创新、投融资、上下游图谱。
  - 产业资讯与动态：政策动向、融资报道、技术突破、AI 周热点。
  - 政策文件检索：政策条文、申报条件、认定标准、政策解读。
  - 产业报告查阅：年度报告、运行分析、调研报告的关键信息提取。
  - 数据计算与分析：同比、环比、占比、贡献率、拉动点数、趋势分析。
  - 图表与可视化：折线图、柱状图、饼图、分组图、堆叠图（基于真实数据）。
  - 报告撰写与导出：分析报告生成，导出 Word / Excel。
  - 文件附件分析：PDF/Word/Excel/图片上传后摘要、问答、对比、异常识别。
  - 互联网信息补充：内部无数据时补充公开信息并提示可靠性差异。
- `可直接提问模板（示例）`：
  - 数据查询类：
    - `2024年规上工业增加值是多少？`
    - `2025年1-3月规上工业总产值同比增速是多少？`
    - `近五年工业增速走势如何？`
  - 产业链分析类：
    - `介绍一下新能源汽车产业的总体发展情况。`
    - `AI产业有多少企业、有哪些上市公司？`
    - `机器人产业近五年融资趋势是上升还是下降？`
  - 政策与报告类：
    - `支持工业企业数字化转型有哪些政策？`
    - `工业技改资金补贴的申请条件是什么？`
    - `帮我总结知识库里智能制造报告的核心结论。`
  - 计算与可视化类：
    - `帮我计算2024年高技术制造业占规上工业比重。`
    - `画一个2020-2024年规上工业增加值折线图。`
  - 输出与文件类：
    - `基于以上分析生成一份Word报告并导出。`
    - `[上传文件] 帮我对比这两份报告的主要差异。`
- `提问技巧`：
  - 明确时间：`2024年`、`2025年1-3月`、`近五年`。
  - 明确口径：`规模以上工业`、`高技术制造业`、`战略性新兴产业`。
  - 明确目标：`查数值` / `做分析` / `画图` / `导出` / `解读政策`。
  - 复杂任务分步：`先查数据 -> 再分析 -> 再画图或导出`。
- `边界与可信度说明`：
  - 无数据时必须明确告知，禁止编造。
  - 关键结论尽量附来源（数据库/知识库/公开链接）。
  - 内部数据优先于互联网信息；互联网用于补充最新动态。
  - 政策可能存在更新时滞，需要提醒用户关注最新发布节点。

## Response Template
1. `你可以问的能力范围（10类）`
2. `你可以直接复制的问题（8-12条）`
3. `提问建议（时间+口径+目标）`
4. `边界说明（数据与时效）`
5. `下一步建议（给1条可立即执行的追问）`
