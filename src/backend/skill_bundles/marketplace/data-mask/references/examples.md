# 数据脱敏使用示例集

> 本文档展示从触发到输出的完整使用流程，覆盖各文件格式和策略组合。
> 每个示例包含：用户触发语 → 操作命令 → 执行效果。

---

## 示例 1：纯文本脱敏（最常用场景）

### 用户触发语
```
帮我把这段文字的数字脱敏：
"公司2025年实现营业收入158000万元，同比增长23.5%，净利润32000万元。"
```

### 执行命令
```bash
echo "公司2025年实现营业收入158000万元，同比增长23.5%，净利润32000万元。" | python scripts/mask.py -s hash
```

### 输出效果
```
公司8729年实现营业收入443350万元，同比增长77%，净利润30000万元。
```
> 数字已替换但仍保持数据形态，可用于 AI 分析。如需还原，保存映射表后执行 `--unmask`。

## 示例 2：Excel 表格脱敏（列级精准控制）

### 用户触发语
```
把这几个城市的分城市运营数据 Excel 脱敏，只脱敏订单和收入列，城市名不动。
```

### 执行命令
```bash
python scripts/dataframe_mask.py -i city_data.xlsx -c "日均订单,总收入" -m city_map.json
```

### 输出效果

| 城市 | 日均订单（脱敏前） | 日均订单（脱敏后） | 总收入（脱敏前） | 总收入（脱敏后） |
|------|------------------|------------------|----------------|----------------|
| 北京 | 853,000 | 720,000 | 1,580 万元 | 4,433 万元 |
| 上海 | 621,000 | 700,000 | 1,200 万元 | 3,000 万元 |
| 广州 | 487,000 | 195,000 | 890 万元   | 2,100 万元 |

> 城市名列保留，数值列精准脱敏。映射表 `city_map.json` 记录了所有替换关系。

## 示例 3：Word 文档脱敏（保留全部格式）

### 用户触发语
```
这份运营月报 Word 文档里面有表格和图表里的数据，帮我全篇脱敏，格式不要动。
```

### 执行命令
```bash
python scripts/file_mask.py -i 运营月报.docx -m monthly_map.json
# 输出：运营月报_mask.docx（自动命名）
```

### 执行效果

| 检查项 | 结果 |
|-------|------|
| 正文段落数字 | ✅ 已替换 |
| 表格内数字 | ✅ 已替换 |
| 页眉页脚数字 | ✅ 已替换 |
| 字体/字号/颜色 | ✅ 保留 |
| 表格结构/样式 | ✅ 保留 |
| 图片 | ✅ 保留（图片内数字通过 OCR 可选处理） |

## 示例 4：PPT 演示文稿脱敏

### 用户触发语
```
下周要做汇报，把这 PPT 里的数字都模糊化，让数据趋势还在但看不到真实值。
```

### 执行命令
```bash
python scripts/file_mask.py -i 汇报.pptx -s blur
# 输出：汇报_mask.pptx
```

### 执行效果

| 幻灯片页 | 脱敏前 | 脱敏后 | 效果 |
|---------|--------|--------|------|
| 第3页 标题 | 2025年GMV突破500亿 | 2025年GMV突破518亿 | 数字真实感保留 |
| 第5页 表格 | 转化率 3.2% | 转化率 3.5% | 趋势保留，精确值模糊 |
| 第8页 图表数据 | 环比增长 12.8% | 环比增长 13.1% | 增长趋势不变 |

> blur 策略保留数据趋势，适合内部分享场景。

## 示例 5：图片 OCR 脱敏

### 用户触发语
```
这张数据看板截图里有很多数字，帮我把图里的数字脱敏成文本。
```

### 执行命令
```bash
python scripts/file_mask.py -i dashboard.png --ocr-lang chi_sim+eng -o dashboard_masked.txt
```

### 输出
```
原始图片文字识别并脱敏后输出为纯文本文件。
格式：保留段落结构，数字已替换。
```
> ⚠️ 前置条件：需已安装 Tesseract OCR 引擎。首次使用参考主文档「方式8」的安装指引。

## 示例 6：批量处理多个文件

### 用户触发语
```
这个目录下所有 Excel 文件都要脱敏，每个保存一份映射表。
```

### 执行命令
```bash
for f in *.xlsx; do
    python scripts/dataframe_mask.py -i "$f" -m "${f%.xlsx}_map.json"
done
```

### 输出
```
data1.xlsx → data1_mask.xlsx + data1_map.json
data2.xlsx → data2_mask.xlsx + data2_map.json
data3.xlsx → data3_mask.xlsx + data3_map.json
```
