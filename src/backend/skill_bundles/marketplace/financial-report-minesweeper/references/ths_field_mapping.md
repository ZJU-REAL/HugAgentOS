# THS API 字段 → 排雷模板字段映射表

## 数据源说明

三大报表数据通过 akshare 同花顺 API 获取，均为**合并报表口径**：
- 资产负债表：`ak.stock_financial_debt_ths(symbol, indicator="按报告期")`
- 利润表：`ak.stock_financial_benefit_ths(symbol, indicator="按报告期")`
- 现金流量表：`ak.stock_financial_cash_ths(symbol, indicator="按报告期")`

数据免费、无需 API Key、akshare 纯 Python 封装。

## 使用说明

1. THS API 返回的数据，金额以"亿"为单位带单位字符串（如"1662.20亿"），脚本需去除"亿"转为浮点数
2. 标注【无公开数据】的科目直接填写【无公开数据】
3. THS 中值为 `False` 或 `nan` 的科目，如果公司业务中确实无此科目（如大秦铁路无商誉），填写为 0 
4. 合并报表验证：三张表均有"少数股东"相关科目即为合并报表
