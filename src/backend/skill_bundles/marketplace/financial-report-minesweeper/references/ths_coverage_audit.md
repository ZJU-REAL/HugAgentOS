# THS 数据覆盖审计报告

## 审计方法
逐条对照排雷技能 SKILL.md 中模块一至模块六全部指标的计算公式所需字段，与 THS API 实际返回的列名逐一比对。三类判定：
- ✅ **完全可用**：THS 提供全部所需字段，无需附注
- ⚠️ **部分可用**：核心计算可行，但个别字段需附注补充或近似替代
- ❌ **不可用**：关键字段 THS 不提供，必须依赖附注/PDF

## 总结

### 实际影响评估

排雷分析分为两个层面：
- **定量层**（50个保留指标）：THS 覆盖 100%（全部可产出计算结果）
- **定性层**（舞弊红线中的附注依赖项）：审计意见为唯一完全依赖附注的红线，需人工核查年报

### 已移除指标清单

以下指标已从排雷框架中移除，原因及THS实际覆盖情况如下：

**2026-05-28（首批5项）**：
1. **1.7 非经营性资金占用** — THS提供「其他应收款」「预付款项」余额，但无法区分关联方×非经营性分类，纯附注依赖
2. **3.2 固定资产成新率** — THS提供固定资产净值✓，但固定资产原值需附注，公式不可计算
3. **3.4 投资性房地产** — THS提供余额✓，但计量模式（成本/公允价值）纯附注依赖
4. **3.6 未并表可疑主体** — THS提供「长期股权投资」余额，但持股比例/控制权/盈利全部依赖附注
5. **3.7 合并范围完整性** — 纯附注/审计报告依赖

**2026-05-29（第二批3项，THS不提供对应字段）**：
1. **存货跌价准备计提比例** — 原为信号4，THS不单独提供；改为3信号判定，移除该信号
2. **研发资本化金额** — THS不提供；原5.2.2节已移除，研发费用率仍保留在5.2.1
3. **在建工程转固金额** — THS不提供；改为余额增速+占比联合判定替代

### 建议处理方案

1. **⚠️ 指标**：在报表模板中标注【无公开数据，需人工核查附注】，仍计算可计算部分
2. 定量排雷产出不受影响，所有保留指标均可产出至少部分计算结果
