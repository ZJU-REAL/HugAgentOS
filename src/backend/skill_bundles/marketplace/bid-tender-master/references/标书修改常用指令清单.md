# 标书修改常用指令清单

> 用户直接复制使用，不用记复杂命令

---

## 润色类

| 你想要的 | 直接说 |
|---------|--------|
| 让文字更正式 | 优化这段话，写得更正式严谨 |
| 删废话 | 精简这段内容，保留核心意思 |
| 改病句 | 修正本段错别字和语句问题 |
| 改表述 | 修改这句话，不要口语化 |

## 内容修改类

| 你想要的 | 直接说 |
|---------|--------|
| 替换一句话 | 把「XXX」改成「XXX」 |
| 补充内容 | 在本段末尾补充一段：XXX |
| 删除内容 | 删除本段中「XXX」这句话 |
| 全局替换 | 把全文所有「旧名称」统一改为「新名称」 |

## 数据修改类

| 你想要的 | 直接说 |
|---------|--------|
| 改报价 | 总报价修改为 42800 元 |
| 改工期 | 供货期从7天改成5天 |
| 改质保 | 质保期从1年改为3年 |
| 改表格数字 | 把报价表第2项单价改成XXX |

## 表格修改类

| 你想要的 | 直接说 |
|---------|--------|
| 新增业绩 | 在业绩表新增一条：项目名称XXX、甲方XXX、金额XXX、时间XXX |
| 删除人员 | 删除人员表第3行 |
| 修改表格内容 | 把报价表备注改成XXX |

## 章节调整类

| 你想要的 | 直接说 |
|---------|--------|
| 整章重写 | 完整重写【售后服务方案】章节 |
| 扩充内容 | 扩充【质量保障措施】，增加2条管控细则 |
| 简化 | 简化【施工组织设计】，保留核心流程 |

## 回滚类

| 你想要的 | 直接说 |
|---------|--------|
| 改错了 | 撤销上一次修改 |
| 想从头来 | 恢复初始版本 |

## 自查类

| 你想要的 | 直接说 |
|---------|--------|
| 全项检查 | 对标书做全项自查 |
