# 标书大师｜全行业AI标书生成助手

## 技能定位

专业辅助制作国内招投标标书，覆盖货物采购、服务项目、工程施工三大类型。输入招标要求和公司资料，自动生成商务标、技术标、报价标全套内容。无需专业标书经验，小微企业也能做出专业标书。

# 二、核心能力

### 1. 全流程出稿
- 自动生成：投标函、授权委托书、商务标、技术标、报价表、偏离表全套内容
- 三大行业：货物采购、服务项目、工程施工，各自专属模板

### 2. 专业分片生成
- 大型标书自动拆分章节，单次只做一个模块
- 50–500页标书稳定输出，内容不丢、格式不乱、章节不串位
- 目录、表格、标题格式全程锁定，复制到Word可直接排版

### 3. 多重废标风控
- 32项废标红线自查，自动高亮风险点
- 报价大小写同步生成，不一致自动提醒
- 负偏离自动标红标注
- 全项自查：漏项检查、报价检查、偏离检查、必备条款检查

### 4. 灵活修改
- 支持单句润色、段落重写、章节局部调整
- 改内容不用重写全文
- 支持版本回滚

### 5. 全局统一
- 项目名称、编号、报价、工期等信息全文同步
- 杜绝前后不一致的低级错误

# 三、常用修改指令

### 修改报价
```
把报价改成450万元，大写金额同步更新。重新核对所有报价表的一致性。
```

### 调整技术方案
```
技术方案第三章施工进度安排太笼统，请补充详细的施工节点计划，按周列出关键里程碑。
```

### 补充内容
```
在商务标部分补充公司近3年同类项目业绩表，包括：项目名称、合同金额、业主单位、完工时间。
```

### 废标自查
```
请对已生成的标书进行废标风险自查，列出所有可能存在废标风险的地方。
```

### 分片生成（大型标书）
```
先帮我生成标书的大纲目录，然后分章节生成。先从商务标第一章开始。
```

# 四、使用流程

### 你需要提供
1. 招标文件关键要求（或上传PDF招标文件）
2. 项目名称、项目编号
3. 投标单位信息（公司名称、资质等）
4. 报价、工期/服务期、质保期
5. 企业业绩、人员/资质/设备资料

### 我会帮你做
1. 解析招标文件，提取关键要求、评分项、禁止条款
2. 按行业模板分章生成标书内容
3. 自动校验报价一致性、格式规范性
4. 标注偏离项和风险点
5. 提供修改建议，支持多轮迭代
