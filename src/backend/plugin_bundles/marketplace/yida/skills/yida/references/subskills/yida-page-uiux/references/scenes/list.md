# 场景：列表 / 管理页

> 面向「找到 → 筛选 → 处理」某类对象的日常操作台。数据是主角，装饰让位于可读与可操作。

## 场景定位

- **用户目标**：在一堆同类对象里快速定位、批量处理、下钻看详情。
- **不是**：看板（那是聚合结论）、门户（那是入口）、单对象详情（那是 detail）。
- **典型业务**：订单管理、客户列表、工单池、审批记录、商品管理。

## 布局骨架 / 信息架构

```
┌─────────────────────────────────────┐
│ 顶部：标题 + 主操作（新建）           │
├─────────────────────────────────────┤
│ 筛选栏（常用筛选平铺，高级筛选收起）   │  ← 克制一行，别堆一屏筛选
├─────────────────────────────────────┤
│ 数据表 / 卡片列表（主次列宽）         │  ← 焦点，密度高
│   行内轻操作（查看/编辑/更多）        │
├─────────────────────────────────────┤
│ 分页 / 加载更多                       │
└─────────────────────────────────────┘
         点行 → 详情抽屉（不跳页）
```

移动端：表格退化为卡片列表（每行一卡，关键字段 2-3 个），筛选收进顶部抽屉。

## 信息密度

**高**。列表天生密集，靠**列宽主次 + 行高节奏 + 对齐**建立可读秩序，而不是靠留白稀释。金额/数量/状态等关键列要一眼可扫。

## 焦点（hero）元素

**数据表本身**。筛选栏、分页、操作都是配角，视觉重量给到行数据。主键列（名称/编号）最重，次要字段弱化。

## 组件套餐（映射 design-system.md / component-jsx-guide.md）

- 表格：`design-system.md` 紧凑表格 / 数据列表行；主键列加重，次要列弱化文字色
- 状态：`design-system.md` 标签（tag），语义色（告警红仅给逾期/失败等真异常）
- 筛选：自定义下拉（**禁 `<select>`**，见 `yida-custom-page` FATAL 规则）+ 搜索框 + 时间范围
- 行操作：图标按钮（功能性图标，克制）+ 「更多」收起长尾操作
- 详情：**抽屉（drawer）而非跳页**，保留列表上下文；需录入则内嵌 `iframe workbench/{formUuid}?iframe=true`
- 空/错/载态：`design-system.md` 空状态 / 骨架屏 / 错误态**三态齐全**（列表最容易漏空态）

## 本场景专属去 AI 味要点

- ❌ **禁每列等宽平铺**：所有列一样宽 = 没有主次。主键列宽、时间/状态列窄，按信息重要度分配列宽。
- ❌ **禁行内塞满彩色标签**：每个字段都上色 = 花。只有状态/告警用色，其余用中性字色分层级。
- ❌ **禁点行跳整页**：下钻用抽屉保留列表上下文，别让用户「看完一条要退回来重新筛」。
- ✅ **三态必须齐全**：空态（无数据友好提示 + 新建入口）、加载态（骨架屏非转圈）、错误态（可重试）。
- ✅ **筛选克制**：常用筛选平铺一行，高级筛选收进「更多筛选」，别一屏全是筛选控件。
- ✅ **批量操作有反馈**：勾选后出现批量操作条，操作后有结果提示（成功数/失败数）。

## 视觉方向决策块示例

```markdown
### 【视觉方向决策】
- 页面类型：list（用户诉求是"订单管理，能筛选、看状态、点进去处理"）
- 气质关键词：高效、清晰、可操作
- 项目特定原则：① 订单状态列必须一眼可扫（待付款/已发货/已完成/退款）；② 告警红只给「退款/超时未发货」；③ 金额列 tabular-nums 右对齐
- 布局骨架：标题+新建 → 筛选栏(状态+时间+搜索) → 数据表(订单号/客户/金额/状态/时间/操作) → 分页；点行开详情抽屉
- 信息密度：高（列宽主次 + 行高节奏建立秩序）
- 视觉焦点：数据表（订单号列最重，金额右对齐）

### 【差异化 5 维】
1. 辅助/强调色：状态用语义色阶（待办=点缀色、完成=中性、异常=告警红）；避免每状态都高饱和
2. 中性冷暖偏色：冷灰基底（管理工具专业感）
3. 圆角性格：微圆（表格容器微圆，行不加圆角，克制）
4. 排版性格：主键列 600 加重，次要列弱化文字色；金额 tabular-nums 右对齐
5. 装饰母题：选中行左侧主色竖条 + 表头细分隔线（非满行描边）

### 【反默认说明】
用列宽主次 + 中性字色分层替代「等宽列 + 满行彩色标签」，下钻用抽屉非跳页——3 维偏离默认列表脸。

### 【图标策略】
内联 SVG，仅行操作（查看/编辑/更多）和状态用图标；表头、字段名不配装饰图标。
```
