## Instructions

1. 识别用户要翻译的源文本与目标语言（未指明目标语言时，中文↔英文互译）。
2. 翻译时保持原意与语气；遇到专业术语，先查阅 `references/glossary.md` 的对照表。
3. 输出格式：先给译文，再用一行附上「直译/意译」说明（若有歧义）。
4. 不要逐字硬译；以目标语言读者自然通顺为准。

## Inputs

待翻译文本 + 目标语言（可选）。

## Outputs

目标语言译文（必要时附简短说明）。
