# 站点（对话建站与平台托管）

> 最后更新：2026-07-13

「站点」让用户在对话里描述需求，由智能体生成完整的静态网站并一键发布——平台直接托管，任何人凭链接即可访问，之后还能继续通过对话迭代更新。整条链路：沙箱内生成多文件站点 → `publish_site` 工具发布 → nginx 反代后端托管路由对外服务。

站点功能属**社区版（CE）**，入口在「实验室」（需管理员开通实验室权限）。

## 使用方式

1. 在对话里直接提出需求，例如「帮我做一个产品介绍网站并发布」。智能体会在沙箱里生成完整的静态站（HTML/CSS/JS/图片，须有 `index.html` 入口），然后调用 `publish_site` 工具发布，并把访问地址（形如 `/site/<slug>/`）交付到对话里。
2. 打开「实验室 → 站点」可管理自己的全部站点：打开、复制链接、修改标题 / 访问地址 / 可见性、删除。
3. 要修改已发布的站点，回到对话继续描述改动即可——智能体带 `site_id` 重新发布，URL 不变、版本号 +1。

## 可见性

| 档位 | 行为 |
|---|---|
| 公开（默认） | 任何人凭链接访问，无需登录 |
| 团队 | 指定团队的成员登录后可见 |
| 私密 | 仅站点所有者本人登录后可见 |

## 版本与回滚

每次发布生成一个不可变版本目录，线上 URL 原地切换。「站点管理 → 版本」可查看历史版本并一键回滚；回滚后再次发布，版本号接历史最大值递增。

## 轻后端能力（KV 与表单）

站点不止是静态页——平台内置两类站内 API（站内 JS 用相对路径 fetch，无需鉴权配置）：

- **KV 存储**（计数器、游戏分数、轻量配置）：`GET/PUT/DELETE __api/kv/<key>`，value ≤ 4KB、每站 ≤ 200 键；
- **表单收集**（留言、报名、反馈）：`POST __api/forms/<form_key>`（JSON，≤ 8KB/条、≤ 5000 条/站），站主在「站点管理 → 表单数据」查看、清空，或一键**导出 CSV 到「我的空间」**。

`__api/` 为保留前缀（站点文件不可占用）；写操作有限流保护。

## 访问统计

平台统计每个站点的 HTML 页访问次数（资源文件不计），在站点卡片与「分享记录」页展示；已发布站点同时出现在「分享记录」页顶部区块，便于统一管理对外链接。

## 托管与安全

- 站点文件按版本存储在平台存储后端（`sites/<site_id>/v<版本>/`），本地与云存储（S3/OSS）模式均支持；发布新版即时生效，历史版本保留（本地模式保留最近 3 版）。
- 访问路径 `/site/<slug>/…` 由 nginx 原样反代到后端公开托管路由，由后端控制可见性、缓存与安全响应头。
- 公开站点响应带 `Content-Security-Policy: sandbox`（不含 `allow-same-origin`）：站内脚本以独立源运行，无法携带访问者登录态调用平台接口；相应地站内 `localStorage` / cookie 不可用。
- 所有站点响应带 `X-Robots-Tag: noindex`，不进搜索引擎。

## 限制

- 站点本体为静态文件；动态能力限于内置 KV 与表单收集 API，不支持自定义服务端逻辑。
- 单站 ≤ 300 个文件、总量 ≤ 30MB、单文件 ≤ 10MB；单用户 ≤ 50 个站点。
- 内网部署时外部 CDN 资源可能不可达，样式脚本建议内联或本地化（智能体默认遵循此约定）。

## 关键实现位置

| 环节 | 位置 |
|---|---|
| 发布工具 | `src/backend/core/llm/tools/site_tool.py`（`publish_site`） |
| 业务服务 | `src/backend/core/services/site_service.py` |
| 公开托管路由 | `src/backend/api/routes/sites_serve.py`（`GET /site/{slug}/{path}`） |
| 管理接口 | `src/backend/api/routes/v1/sites.py`（`/v1/sites`） |
| 数据表 | `sites`（`core/db/models/site.py`） |
| 前端管理面板 | `src/frontend/src/components/sites/SitesPanel.tsx`（实验室 → 站点） |
| nginx 转发 | `src/frontend/default.conf.template` 的 `location /site/` |

环境开关：`SITES_ENABLED=false` 可整体关闭发布工具（默认开启）。
