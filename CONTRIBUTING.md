# 参与贡献

感谢关注 HugAgentOS 社区版！

本仓库分两类内容：

1. **`src/**`、`mcp_servers/**` 等生成代码** —— 由上游主仓按版本自动生成、
   对外只读。直接对这些文件提 PR 无法被合并；请改提 Issue（附复现/补丁），
   维护者确认后回流主仓，并随下一个版本发布。
2. **`document/`、`examples/`、`README.md`** —— 欢迎直接提 PR。

## 提 Issue

- Bug：请附复现步骤、期望行为、实际行为、环境信息（OS / Docker / Python 版本）。
- 功能建议：请说明使用场景与预期收益。

## 安全问题

请勿公开披露安全漏洞，参见 [SECURITY.md](SECURITY.md)。
