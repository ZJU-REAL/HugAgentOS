# 图标资产清单

本技能内置了项目 UI 设计稿中的全部 SVG 图标，位于 `assets/` 目录。
当需要在前端使用图标时，优先从此清单中选取，保持视觉一致性。

## 使用方式

将 SVG 内容内联到 React 组件中，或复制到项目的 `src/frontend/public/` 或 `src/frontend/src/assets/` 目录后通过 import 引用。

```tsx
// 方式一：内联 SVG（推荐小图标）
function SendIcon() {
  return (
    <svg width="34" height="34" viewBox="0 0 34 34" fill="none">
      {/* ... SVG path ... */}
    </svg>
  );
}

// 方式二：img 标签引用
import sendIcon from '../assets/icons/send.svg';
<img src={sendIcon} alt="发送" />
```

## 设计约束

- **主色**: 图标主色为 `#126DFF`（蓝色），与设计规范一致
- **灰色**: 非激活态图标用 `#808080`
- **尺寸**: 常用 16px / 28px / 32px / 34px / 38px（均为 4 的倍数或接近）
- **风格**: 线性为主，部分填充，圆角端点
- 详见 `ui-design-spec.md` 第 3 节「图标」规范
